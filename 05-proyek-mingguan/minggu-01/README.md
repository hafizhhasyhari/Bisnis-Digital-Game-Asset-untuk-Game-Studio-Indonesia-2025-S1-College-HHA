Minggu ini, kita mengerjakan 4 komponen proyek yang membangun fondasi.

### 1. (P1-Bisnis) Analisis Marketplace
* **Tugas:** Buat file Markdown (`analisis-pasar.md`) di folder ini.
* **Isi:** Buat tabel perbandingan sederhana untuk 3 platform: **Unity Asset Store, Sketchfab, dan Unreal Marketplace**.
* **Kriteria:** Bandingkan (minimal):
    * Potongan Komisi (Berapa % yang mereka ambil?)
    * Tipe Aset Utama (Apa yang paling laku di sana?)
    * Target Audiens (Siapa pembelinya?)

### 2. (P2-CSP) Sketsa Aset UI 2D
* **Tugas:** Buat 5 ikon UI (User Interface) sederhana.
* **Tools:** Gunakan Clip Studio Paint (atau software gambar lain).
* **Tema:** Ikon game dasar (Contoh: Hati (nyawa), Koin (uang), Pedang (serang), Perisai (bertahan), Potion (obat)).
* **Format:** Simpan sebagai `.png` transparan (512x512 px).
* **Commit:** Upload file ke `08-karya-seni-ip/sketsa-ui/`.

### 3. (P3-Blender) Aset Low-Poly Pertama
* **Tugas:** Buat 1 model 3D *low-poly* sederhana.
* **Tools:** Gunakan Blender.
* **Tema:** Properti *background* (Contoh: Peti, Tong, Cangkir, Buku).
* **Batasan:** Jaga *polycount* tetap rendah (di bawah 500 *faces*).
* **Commit:** Upload file `.blend` Anda ke folder ini (misal: `peti.blend`).

### 4. (P4-Konsep) Elevator Pitch Proyek Akhir
* **Tugas:** Tulis *elevator pitch* (1 paragraf) untuk ide proyek akhir Anda.
* **Format:** "Game/Aset Pack saya adalah [Deskripsi Singkat], yang ditujukan untuk [Target Audiens], yang memecahkan masalah [Masalah] dengan [Fitur Unik]."
* **Contoh:** "Saya akan membuat *Asset Pack 3D Low-Poly* tema 'Pasar Terapung Indonesia', ditujukan untuk *developer game indie* yang ingin membuat game *cozy* dengan nuansa Asia Tenggara yang unik."
* **Commit:** Buat file `elevator-pitch.md` di folder ini dan tulis *pitch* Anda.

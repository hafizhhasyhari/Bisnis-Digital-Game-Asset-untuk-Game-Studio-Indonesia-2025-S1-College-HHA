# ⚙️ Praktik Lab Minggu 01: Setup Workflow & Moodboard

## Tujuan

1.  Mengonfigurasi *tools* (Blender & CSP) untuk *workflow* yang efisien.
2.  Membuat *moodboard* visual untuk ide proyek akhir semester.

## Bagian 1: Setup Tools (Rekomendasi)

### Blender
1.  **Instalasi:** Pastikan Anda menggunakan versi LTS (Long Term Support) terbaru atau versi stabil terbaru.
2.  **Essential Add-ons:**
    * Buka `Edit > Preferences > Add-ons`.
    * Aktifkan:
        * `Node: Node Wrangler` (Wajib untuk *shading*).
        * `Import-Export: Import Images as Planes` (Sangat berguna untuk *reference*).

### Clip Studio Paint (CSP)
1.  **Instalasi:** Gunakan versi Pro atau EX.
2.  **Atur Workspace:**
    * Buka `Window` dan pastikan panel `Sub-View` dan `Material` Anda aktif.
    * Panel `Sub-View` sangat baik untuk memuat gambar referensi tanpa mengotori kanvas utama.
3.  **Download Aset (Opsional):**
    * Buka `Clip Studio ASSETS` (logo awan).
    * Cari *brush* atau material gratis yang Anda sukai (misal: *pencil brush*).

## Bagian 2: Praktik Moodboard

Sebuah *moodboard* adalah kolase visual yang membantu mendefinisikan *feel* dan *look* dari proyek Anda.

1.  **Tentukan Tema:** Pilih tema untuk proyek akhir Anda (contoh: "Sci-Fi Indonesia", "Cozy Fantasy", "Horror Low-Poly").
2.  **Cari Referensi:** Buka Pinterest, Google Images, atau ArtStation. Kumpulkan 5-10 gambar yang mewakili:
    * Palet Warna
    * Arsitektur / Lingkungan
    * Gaya Karakter
    * *Feel* (Emosi)
3.  **Buat Kanvas:** Buka CSP. `File > New`. Set resolusi ke `1920 x 1080 px` @ 72 dpi.
4.  **Impor & Susun:**
    * *Drag-and-drop* gambar-gambar referensi Anda ke dalam kanvas CSP.
    * Atur ukurannya, potong, dan susun hingga terlihat menarik.
    * **Tips:** Jangan tumpang tindih terlalu banyak. Beri "ruang napas".
5.  **Simpan & Ekspor:**
    * Simpan file kerja Anda (`.csp`).
    * Ekspor sebagai JPG/PNG (`File > Export (Single Layer) > .png`).
6.  **Pamerkan!**
    * Upload file `.png` Anda ke `04-pameran-showcase/minggu-01/`.
    * (Opsional) Bagikan di media sosial Anda dengan *hashtag* [HashtagAnda].
